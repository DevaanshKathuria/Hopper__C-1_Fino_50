# Final Submission Checklist

- Repository structure is clean and up to date.
- Raw datasets are present and documented.
- ETL scripts run successfully and generate reproducible processed outputs.
- EDA notebooks are cleaned, readable, and saved with final outputs.
- Statistical analysis steps are documented and reproducible.
- Tableau-ready datasets are exported and versioned where appropriate.
- Tableau dashboard screenshots are stored in `tableau/screenshots/`.
- Final report is stored in `reports/`.
- Presentation/deck is finalized and reviewed.
- Team contributions are documented.
- README and docs are consistent with the final submission state.
- All claims in the report are supported by code, tables, charts, or dashboards.
